Tasks

This is an example and testing environment for my TODO's application.
The tasks here are stored in your session on the server and gone after a session timeout (30 mins).
 

You can add new tasks by typing a name into the add field. Additionally so called tasks contexts (like groups, categories etc.) can be defined via @categoryname.
 

For instance type: 'My really urgent task @work @london'.
 

No go play around and have fun :-)
